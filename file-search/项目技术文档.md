## 1. 项目概述

### 1.1 项目背景

在Linux服务器环境中，传统的文件检索工具（如`find`命令）存在以下问题：

1. **性能瓶颈**: 遍历文件系统耗时较长
2. **远程访问困难**: 无法通过网络进行远程检索
3. **并发支持不足**: 不支持多用户同时检索
4. **缺乏索引机制**: 每次检索都需要重新遍历文件系统

本项目旨在解决这些问题，提供一个高性能、支持远程访问、多用户并发的文件检索服务。

### 1.2 项目目标

- **高性能**: 检索响应时间 ≤ 1秒（1000个文件）
- **高并发**: 支持最多20个客户端同时连接
- **实时更新**: 每5分钟自动更新文件索引
- **远程访问**: 通过TCP协议提供远程检索服务
- **持久化存储**: 使用MySQL数据库存储索引和检索记录

### 1.3 技术栈

## 5. 算法设计

### 5.1 二叉搜索树算法

#### 5.1.1 插入算法

```c
TreeNode* insert_tree_node(TreeNode *root, TreeNode *new_node) {
    // 空树，返回新节点
    if (root == NULL) {
        return new_node;
    }
    
    // 比较文件名
    int cmp = strcmp(new_node->filename, root->filename);
    
    if (cmp < 0) {
        // 插入左子树
        root->left = insert_tree_node(root->left, new_node);
    } else if (cmp > 0) {
        // 插入右子树
        root->right = insert_tree_node(root->right, new_node);
    }
    // cmp == 0 时，文件名已存在，不插入
    
    return root;
}
```

**时间复杂度**: O(log n) 平均，O(n) 最坏

#### 5.1.2 搜索算法

```c
TreeNode* search_tree(TreeNode *root, const char *filename) {
    if (root == NULL) {
        return NULL;
    }
    
    int cmp = strcmp(filename, root->filename);
    
    if (cmp == 0) {
        return root;  // 找到
    } else if (cmp < 0) {
        return search_tree(root->left, filename);  // 搜索左子树
    } else {
        return search_tree(root->right, filename);  // 搜索右子树
    }
}
```

**时间复杂度**: O(log n) 平均，O(n) 最坏

#### 5.1.3 遍历算法

```c
void traverse_tree(TreeNode *root, SearchResult *result, SearchCondition *cond) {
    if (root == NULL) {
        return;
    }
    
    // 中序遍历：左-根-右
    traverse_tree(root->left, result, cond);
    
    // 检查当前节点是否符合条件
    if (check_condition(root, cond)) {
        add_to_result(result, root);
    }
    
    traverse_tree(root->right, result, cond);
}
```

**时间复杂度**: O(n)

### 5.2 文件索引构建算法

```c
int build_file_index(const char *base_dir) {
    DIR *dir = opendir(base_dir);
    if (dir == NULL) {
        return -1;
    }
    
    struct dirent *entry;
    int file_count = 0;
    
    while ((entry = readdir(dir)) != NULL) {
        // 跳过 "." 和 ".."
        if (strcmp(entry->d_name, ".") == 0 || 
            strcmp(entry->d_name, "..") == 0) {
            continue;
        }
        
        // 构建完整路径
        char full_path[1024];
        snprintf(full_path, sizeof(full_path), 
                 "%s/%s", base_dir, entry->d_name);
        
        // 检查路径合法性
        if (!check_path_valid(full_path)) {
            continue;
        }
        
        // 获取文件信息
        struct stat file_stat;
        if (stat(full_path, &file_stat) == 0) {
            if (S_ISREG(file_stat.st_mode)) {
                // 普通文件：创建节点并插入BST
                TreeNode *node = create_tree_node(
                    entry->d_name, 
                    full_path,
                    file_stat.st_size, 
                    file_stat.st_mtime
                );
                
                pthread_mutex_lock(&g_index_mutex);
                g_file_index_root = insert_tree_node(g_file_index_root, node);
                pthread_mutex_unlock(&g_index_mutex);
                
                file_count++;
            } else if (S_ISDIR(file_stat.st_mode)) {
                // 目录：递归遍历
                file_count += build_file_index(full_path);
            }
        }
    }
    
    closedir(dir);
    return file_count;
}
```

**算法特点**:
- 递归遍历目录树
- 跳过特殊目录（"."、".."）
- 检查路径合法性
- 使用互斥锁保护共享资源

### 5.3 多条件检索算法

```c
int check_condition(TreeNode *node, SearchCondition *cond) {
    int match = 1;
    
    // 精确匹配文件名
    if (match && cond->has_filename) {
        if (strcmp(node->filename, cond->filename) != 0) {
            match = 0;
        }
    }
    
    // 模糊匹配文件名
    if (match && cond->has_filename_like) {
        if (strstr(node->filename, cond->filename_like) == NULL) {
            match = 0;
        }
    }
    
    // 文件大小条件
    if (match && cond->has_size_gt) {
        if (node->size <= cond->size_gt) {
            match = 0;
        }
    }
    
    // 修改时间条件
    if (match && cond->has_mtime_gt) {
        if (node->mtime <= cond->mtime_gt) {
            match = 0;
        }
    }
    
    // 正则表达式匹配
    if (match && cond->has_regex) {
        regex_t regex;
        if (regcomp(&regex, cond->regex_pattern, REG_EXTENDED) == 0) {
            if (regexec(&regex, node->filename, 0, NULL, 0) != 0) {
                match = 0;
            }
            regfree(&regex);
        }
    }
    
    return match;
}
```

**算法特点**:
- 短路求值：一旦不匹配立即返回
- 支持多种条件组合
- 使用POSIX正则库

### 6.3 命令解析实现

```c
int parse_search_command(const char *cmd, SearchCondition *cond) {
    memset(cond, 0, sizeof(SearchCondition));
    
    // 检查是否是SEARCH命令
    if (strncmp(cmd, "SEARCH ", 7) != 0) {
        return -1;
    }
    
    const char *p = cmd + 7;
    char token[4096];
    
    while (*p != '\0' && *p != '\n') {
        // 跳过空格
        while (*p == ' ') p++;
        if (*p == '\0' || *p == '\n') break;
        
        // 读取一个条件
        int i = 0;
        while (*p != ' ' && *p != '\0' && *p != '\n' && i < sizeof(token) - 1) {
            token[i++] = *p++;
        }
        token[i] = '\0';
        
        // 解析条件
        if (strncmp(token, "filename=", 9) == 0) {
            cond->has_filename = 1;
            strncpy(cond->filename, token + 9, 255);
        } else if (strncmp(token, "filename_like=", 14) == 0) {
            cond->has_filename_like = 1;
            strncpy(cond->filename_like, token + 14, 255);
        } else if (strncmp(token, "size>", 5) == 0) {
            cond->has_size_gt = 1;
            cond->size_gt = atoll(token + 5);
        } else if (strncmp(token, "mtime>", 6) == 0) {
            cond->has_mtime_gt = 1;
            cond->mtime_gt = atol(token + 6);
        } else if (strncmp(token, "regex=", 6) == 0) {
            cond->has_regex = 1;
            strncpy(cond->regex_pattern, token + 6, 255);
        }
    }
    
    return 0;
}
```

```c
int db_insert_file_index(TreeNode *root) {
    if (root == NULL || g_db_conn == NULL) {
        return -1;
    }
    
    // 构建批量插入SQL
    char query[65536] = "INSERT INTO file_index (filename, path, size, mtime) VALUES ";
    char values[1024];
    int first = 1;
    
    // 中序遍历BST，收集所有节点
    void collect_values(TreeNode *node) {
        if (node == NULL) return;
        
        collect_values(node->left);
        
        if (!first) {
            strcat(query, ",");
        }
        first = 0;
        
        snprintf(values, sizeof(values), 
                 "('%s', '%s', %ld, %ld)",
                 node->filename, node->path, node->size, node->mtime);
        strcat(query, values);
        
        collect_values(node->right);
    }
    
    collect_values(root);
    
    // 执行插入
    if (mysql_query(g_db_conn, query) != 0) {
        write_log("Failed to insert file index: %s", mysql_error(g_db_conn));
        return -1;
    }
    
    return 0;
}
```

#### 7.2.2 从数据库搜索文件

```c
int db_search_files(SearchCondition *cond, SearchResult *result) {
    char query[4096] = "SELECT filename, path, size, mtime FROM file_index WHERE 1=1";
    char where[1024] = "";
    
    // 构建WHERE条件
    if (cond->has_filename) {
        char escaped[512];
        mysql_real_escape_string(g_db_conn, escaped, cond->filename, strlen(cond->filename));
        sprintf(where + strlen(where), " AND filename = '%s'", escaped);
    }
    
    if (cond->has_filename_like) {
        char escaped[512];
        mysql_real_escape_string(g_db_conn, escaped, cond->filename_like, strlen(cond->filename_like));
        sprintf(where + strlen(where), " AND filename LIKE '%%%s%%'", escaped);
    }
    
    if (cond->has_size_gt) {
        sprintf(where + strlen(where), " AND size > %ld", cond->size_gt);
    }
    
    if (cond->has_mtime_gt) {
        sprintf(where + strlen(where), " AND mtime > %ld", cond->mtime_gt);
    }
    
    strcat(query, where);
    
    // 执行查询
    if (mysql_query(g_db_conn, query) != 0) {
        write_log("Failed to search files: %s", mysql_error(g_db_conn));
        return -1;
    }
    
    // 获取结果
    MYSQL_RES *res = mysql_store_result(g_db_conn);
    if (res == NULL) {
        return -1;
    }
    
    // 遍历结果集
    MYSQL_ROW row;
    while ((row = mysql_fetch_row(res)) != NULL) {
        FileInfo file;
        strcpy(file.filename, row[0]);
        strcpy(file.path, row[1]);
        file.size = atoll(row[2]);
        file.mtime = atol(row[3]);
        
        add_to_result(result, &file);
    }
    
    mysql_free_result(res);
    return 0;
}
```


## 9. 性能优化

### 9.1 索引优化

#### 9.1.1 内存索引优先

**策略**: 优先查询内存中的BST，查询不到再查数据库。

**优势**:
- 内存访问速度比磁盘快1000倍以上
- BST查询时间复杂度O(log n)
- 减少数据库压力

**实现**:
```c
void search_files(SearchCondition *cond, SearchResult *result) {
    // 1. 先查内存索引
    pthread_mutex_lock(&g_index_mutex);
    traverse_tree(g_file_index_root, result, cond);
    pthread_mutex_unlock(&g_index_mutex);
    
    // 2. 如果内存索引未找到，查数据库
    if (result->count == 0) {
        db_search_files(cond, result);
    }
}
```

#### 9.1.2 索引更新策略

**异步更新**: 后台线程每5分钟更新一次索引，不阻塞主服务。

```c
void* index_update_thread(void *arg) {
    while (1) {
        sleep(300);  // 5分钟
        
        write_log("Updating file index...");
        
        // 加锁
        pthread_mutex_lock(&g_index_mutex);
        
        // 释放旧索引
        free_tree(g_file_index_root);
        g_file_index_root = NULL;
        
        // 构建新索引
        int file_count = build_file_index(SEARCH_BASE_DIR);
        
        // 解锁
        pthread_mutex_unlock(&g_index_mutex);
        
        write_log("Index updated: %d files", file_count);
    }
    
    return NULL;
}
```

#### 15.1.4 异步I/O

**技术**: 使用epoll提高网络性能

**实现**:
```c
int epoll_fd = epoll_create1(0);
struct epoll_event ev;
ev.events = EPOLLIN;
ev.data.fd = server_sock;
epoll_ctl(epoll_fd, EPOLL_CTL_ADD, server_sock, &ev);

while (1) {
    int nfds = epoll_wait(epoll_fd, events, MAX_EVENTS, -1);
    for (int i = 0; i < nfds; i++) {
        if (events[i].data.fd == server_sock) {
            // 接受新连接
        } else {
            // 处理客户端请求
        }
    }
}
```
